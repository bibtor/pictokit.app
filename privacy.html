PictoKit

Privacy Policy

Last Updated: May 26, 2026

This privacy policy applies to the PictoKit app (referred to as "Application") for macOS, created by Charles Binet ("Service Provider"). The Application is provided as a freemium service intended for use "AS IS".

Information Collection and Use

The Application does not collect, transmit, or store any personal data. Specifically:

  • No personal information is collected or sent to any server.
  • No analytics, telemetry, or usage tracking is performed by the Application.
  • No user accounts or sign-in are required.
  • All user data — including favorites, recents, clipboard history, and settings — is stored locally on your Mac and never leaves your device.

Website Analytics

We collect anonymous usage analytics on the pictokit.app website via TelemetryDeck. No personal data, clipboard content, IP addresses, or device identifiers are collected or stored. See telemetrydeck.com/privacy for details.

Network Requests

The Application connects to the internet solely for its core features. No personal data is transmitted during these requests:

  • Image search: When you search for photos or images, queries are sent to third-party APIs (Unsplash, Pexels, NASA Image Library, Klipy) to fetch results. Only your search terms are transmitted.
  • Library downloads: Icon libraries are downloaded from GitHub when you choose to install them.
  • App updates: The Application checks for updates via a Sparkle update feed hosted on pictokit.app.

No browsing history, device identifiers, or personal information is included in these requests.

Third Party Services

The Application integrates with the following third-party services solely to provide image and icon search functionality:

  • Unsplash — photo search and download tracking (required by Unsplash Terms of Service)
  • Pexels — photo search
  • NASA Image Library — public domain image search (no API key required)
  • Klipy — GIF, sticker, and clip search

These services have their own privacy policies. The Application does not share any personal data with these services beyond the search queries you enter.

Local Data Storage

The following data is stored locally on your Mac and never transmitted:

  • Favorited icons and images
  • Recently used items (auto-deleted after 3 days)
  • Clipboard history (limited to 20 items, auto-deleted after 3 days)
  • App settings and preferences
  • Downloaded icon libraries
  • Custom folder references

Permissions

The Application may request Accessibility permission on macOS. This is used solely to paste selected icons and images into your currently focused application by simulating a keyboard shortcut. This permission is never used for data collection or monitoring.

Children

Since the Application does not collect any personal information from any user, it is safe for use by individuals of all ages. The Application does not knowingly solicit data from or market to children under the age of 13.

Security

The Application uses the macOS App Sandbox and Hardened Runtime to limit its access to only the permissions you explicitly grant. All locally stored data remains within the Application's sandboxed container.

Changes

This Privacy Policy may be updated from time to time. You are advised to review this page periodically for any changes. Changes are effective immediately after they are posted on this page.
